## lombokの使い方
### 使用手順
- 公式からjarをDL
- jarダブルクリックして、IDEにインストール
  - eclipseの場合は、eclipse.iniに起動時に？jarを読むような設定が追加される
- mavenや、gradleなどで依存関係を追加
- ビルド

### 注意点
- ```gradle eclipse``` などでIDE用の設定を既に作成していた場合、認識されない場合があった。
  - ↑の場合は再度同コマンドを実行することで認識。
